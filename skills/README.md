# Skills 管理

由 `npx skills` 管理，技能位于 `~/.agents/skills/`，本目录为链接。

## 安装的技能

| 技能 | 来源 | 用途 |
|------|------|------|
| `find-skills` | vercel-labs/skills | 搜索和发现技能 |
| `spark` | seedex-skills | 头脑风暴 → 生成规格（出 spec 后 STOP） |
| `brainstorming` | obra/superpowers | 头脑风暴 → 出设计文档 |
| `writing-plans` | obra/superpowers | 设计拆成可执行任务计划 |
| `executing-plans` | obra/superpowers | 按计划批量执行，带检查点 |
| `guizang-ppt-skill` | op7418/guizang-ppt-skill | 杂志风/瑞士风 HTML PPT |
| `pdf` | anthropics/skills | PDF 文件处理 |
| `xlsx` | anthropics/skills | Excel 文件处理 |

## 查找技能

```bash
npx skills find <关键词>
```
