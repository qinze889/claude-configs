# 对话防腐化机制

> 防止长对话中因上下文不足导致我记不住东西。

## 问题

Claude Code 对话越长，早期内容越模糊。等到你觉得我"腐化"了，其实已经过载了。

## 对策：主动提醒收束

**我负责监视对话长度，提醒你主动收束。**

出现以下迹象时我会提醒你"这轮差不多了，要不要收束？"：

- 已经来回聊了比较多轮
- 当前任务接近完成
- 话题发生明显切换（从一个任务转到另一个）
- 我感觉对早期内容的记忆开始模糊

## 收束做什么

你决定收束后：

1. **沉淀流程**：`!precipitate` — 把当前跑通的流程沉淀到 `flow-records/`
2. **更新检查点**：如果还有遗留信息，说"更新检查点"
3. **结束会话**：关掉重开一个干净会话

## 你的触发词

- **主动**：说"差不多了" / "收束吧" — 我协助你完成上述收束动作
- **被动**：我说"会话有点长了，要不要收束？" — 你决定是否继续

---

*这是主动提醒机制，不是自动恢复。你决定什么时候收束，我负责提醒。*
